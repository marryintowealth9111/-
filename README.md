## 📌 專案簡介

本專案深度解析全球十大華裔首富的投資思維、理財智慧與財富傳承策略。

與西方富豪不同，華人富豪的財富哲學深植於**儒家文化、家族觀念與風險規避**傳統，同時融合現代金融工具與全球化視野。透過研究他們的決策邏輯，我們可以學習如何在東方智慧與西方方法之間找到平衡。

**您將學到：**
- 華人富豪的「現金為王」與「低買高賣」哲學
- 家族信託與跨世代傳承的實務架構
- 地產、保險、基建的穩健配置策略
- 風險分散與逆週期操作的東方智慧

## 🎯 六大理財支柱

### 1. 現金為王（Cash is King）

- **核心理念**：手上有現金，危機中有機會
- **李嘉誠**：「現金就像氧氣，平時不覺得，缺氧時才知道重要」
- **實務應用**：保持6-12個月生活費的現金儲備，低負債運營

### 2. 風險分散（Don't Put All Eggs in One Basket）

- **核心理念**：跨地域、跨產業、跨週期配置
- **郭鶴年**：糖業、酒店、地產、傳媒——全球佈局，東方不亮西方亮
- **實務應用**：單一資產不超過30%，建立「現金流三頭馬車」

### 3. 逆週期操作（Contrarian Investing）

- **核心理念**：別人恐懼時我貪婪，別人貪婪時我恐懼
- **李兆基**：地產低潮時大舉買入，高點時逐步退出
- **實務應用**：判斷市場週期位置，建立「危機基金」等待機會

### 4. 家族傳承（Family Legacy）

- **核心理念**：財富傳三代，靠的是制度而非個人
- **鄭裕彤**：家族憲章、第三代培養、專業經理人並行
- **實務應用**：家族信託、保險規劃、接班人培養計畫

### 5. 保險槓桿（Insurance Leverage）

- **核心理念**：用保險轉移不可承受的風險，放大傳承效益
- **李嘉誠**：透過保險浮存金獲取低成本資金，投資全球基建
- **實務應用**：終身壽險、年金、長期照護險的組合配置

### 6. 低調隱蔽（Stealth Wealth）

- **核心理念**：財不露白，悶聲發大財
- **郭鶴年**：「大馬糖王」低調經營數十年，外界難窺全貌
- **實務應用**：隱私保護、海外架構、家族辦公室
