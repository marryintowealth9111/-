# 交流指南

感謝您對「十大華裔首富的投資思維、理財智慧」專案的興趣！我們歡迎各種形式的交流與分享，讓華人財富智慧能夠開源共享，惠及更多人。

## 🤝 我們需要什麼樣的交流

### 1. 內容分享

| 類型 | 說明 | 範例 |
|-----|------|------|
| **富豪檔案更新** | 補充最新動態、新增案例 | 李嘉誠最新投資、馬雲近況 |
| **技能文件** | 撰寫六大支柱的詳細教學 | `docs/mindset/cash-is-king.md` |
| **實務模板** | 提供可操作的框架工具 | 家族憲章模板、保險配置模型 |
| **案例分析** | 真實案例（匿名化） | 客戶傳承規劃、爭產調解 |
| **資源推薦** | 書籍、課程、專家 | 新出版的財富傳承書籍 |

### 2. 專業知識

| 領域 | 需求 | 分享方式 |
|-----|------|---------|
| **法律** | 各地信託法、繼承法更新 | 補充法規變化、案例判決 |
| **稅務** | 跨境稅務、節稅策略 | 更新稅率、補充規劃技巧 |
| **保險** | 產品設計、核保理賠 | 補充產品應用場景 |
| **投資** | 資產配置、風險管理 | 補充市場分析、工具推薦 |
| **家族辦公室** | 實務操作、治理結構 | 分享實務經驗、最佳實踐 |

### 3. 翻譯與在地化

- 將簡體中文內容翻譯為繁體中文、英文
- 補充各地華人富豪（台灣、新加坡、馬來西亞、泰國等）
- 補充各地法規差異（中國大陸、香港、台灣、新加坡、美國等）
